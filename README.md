# Arbitrary Precision Calculator (APC)

An Arbitrary Precision Calculator implemented in C that performs arithmetic
operations on very large numbers using linked lists.

## Features
- Addition, subtraction, multiplication, and division
- Handles numbers larger than standard data type limits
- Modular and structured implementation

## Data Structures Used
- Linked List

## Concepts Covered
- Dynamic memory allocation
- Pointer manipulation
- Modular programming

## How to Compile
gcc *.c -o apc

## How to Run
./apc
